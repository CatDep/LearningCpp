# Learning C++

Учебный репозиторий для изучения C++ с нуля до уровня Junior C++ Developer / Junior Unreal Engine C++ Developer.

Здесь я прохожу C++ по долгосрочному плану: от базового синтаксиса и консольных мини-проектов до ООП, STL, CMake, тестов и Unreal Engine C++.

---

## Цель

Получить крепкую базу C++, научиться писать самостоятельные программы, собрать портфолио и постепенно перейти к разработке на Unreal Engine C++.
